Stoel Rives

Challenge

Create proprietary software for a tier 1 company used to ticket and escalate incidents in a privacy suite.

My Roles

UX Designer, Associate Product Owner, and Client Manager

Solution

Step 1: Initial discovery

We worked with the client to discover their goals for this product. Working backwards we defined all the requirements that needed to be met.

Step 2: User Stories

Working directly with the client we created user stories. From these we created our feature list.

Step 3: UX Design

I created a clean and simple interface for interacting with the privacy suite. The goal was to make a clear indication of the tasks that needed to be performed and the urgency of current incidents. I used Adobe Illustrator to create mockups for structure and styling.

Step 4: Implementation

I worked directly with the developers to make sure our features worked and the user experience flowed, we finished the project on time with our full feature list.

Step 5: Sunset Reviews

After the project, we had a review both internally and with the client to go over the process and looked at ways to increase our productivity and increase communication.
